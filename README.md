## Property Sales Management System

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Employee, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. House Information
5. Announcement Information
6. Personal Center
7. House Favorites
8. House Viewing Appointment
9. Contract

**Employee Functions:**
1. Login
2. Change Password
3. Personal Information
4. House Management
5. House Comment Management
6. House Viewing Appointment Management
7. Contract Management
8. Announcement Management
9. User Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (House Type Management, Building Management, Announcement Type Management, Contract Type Management)
5. House Management
6. House Comment Management
7. House Favorite Management
8. House Viewing Appointment Management
9. Contract Management
10. Announcement Management
11. User Management
12. Employee Management
13. Carousel Management
